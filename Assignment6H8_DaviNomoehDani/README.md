# Assignment 6 - Kampus Merdeka
CRUD Users Management using Laravel 8 and Bootstrap 4.

## Clone and Run
```sh
$ composer install
$ cp .env.example .env
$ php artisan key:generate
$ php artisan migrate:fresh --seed
$ php artisan serve
```

# Requirements
- Text editor: [Visual Studio Code](https://code.visualstudio.com/)
- Laravel: ^8
- PHP: ^7.3
- Local DBMS
- Browser
