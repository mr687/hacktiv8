# Assignment 6 - Kampus Merdeka
CRUD Users Management using Laravel 8 and Bootstrap 4.

## Clone and Run
```sh
$ composer install
$ cp .env.example .env
$ php artisan key:generate
$ php artisan migrate:fresh --seed
$ php artisan serve
$ http://127.0.0.1:8000/
```
